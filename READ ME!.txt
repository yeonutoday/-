어디까지나 개인의 편의를 위해 기능축소형으로 만들어진 스킨입니다.
※ 개조는 가능. 재배포는 절대불가입니다. (배포는 반드시 절 통해 직접적으로 이루어질수있게 해주세요.)
※ 사용하지않는 기능이 많아 불편하실수있어요. 질문은 홈페이지의 메세지란을 통해 받습니다만,
처음부터 배포용으로 만들어진건 아니라서... 너무 과한 개조에 관련된 복잡,세세한 질문은 지양해주세요.

<< 내용물 >>
nodo_btoolSt_skin.html / nodo_btoolSt_skin.css / image폴더


<< 사용방법 >>
1. nodo_btoolSt_skin.html  의 내용을 복사 -> skin.html 에 붙여넣기.
　nodo_btoolSt_skin.css  의 내용을 복사 -> style.css 에 붙여넣기.
　Image 폴더의 파일을 파일업로드에서 업로드시키기.
2. 화면설정 - 티에디션에서 사용해제.
　화면설정 - 메뉴를 통해 상단 주메뉴(새로고침,태그등) 설정. (이때 이 스킨이 지원하지않는 게스트북/위치로그 등은 메뉴에서 지워주시면 더 좋다!)
　화면설정 - 화면출력에서 선택화면- "내용만"선택.
　화면설정 - 화면출력에서 펼침설정- "댓글펼침"만 선택.
3. 사이드바 - 사이드바1 에 "공지리스트"만.
　사이드바 - 사이드바2에 "검색", "카운터"만.
4. 플러그인 - Light TT EX 사용.
　플러그인 - 가로줄 달력 표시 사용.
5. 기본정보 - 블로그정보 - 블로그설명 에 상단공지처럼 뜰 한마디 넣기.
6. 위의 부분만 반드시 지켜주시면 나머지 설정에 대해서는 융통성있게 조정해주시면됩니다. 


<< 사용주의사항 >>
1. 스킨의 좌측 포스팅영역의 절대넓이는 500px입니다.
　그림이나 동영상 삽입시 500px이 넘어가면 코멘트 영역까지 그림이 넘어가 버리므로 에디터에서 그림사이즈를 조절해서 올려주세요.
　(파일첨부 기능으로 그림파일을 올린뒤, 에디터에서 그림을 클릭, 우측에서 사이즈를 조절하면 클릭시 이미지가 원본사이즈로 커지는 뷰어기능을 이용가능합니다.)
2. 미관상 태그만 노출되어있고 카테고리가 가려져있지만 제대로 기능은 하고있습니다. 편하신대로 이용해주시면 되어요.
3. 사이드바의 추가나 개조등엔 제한을 두고있지않습니다만 하단(푸터)의 카피라이트는 지우지말아주세요.
3. 공지로 글을 작성하시면 상단 주메뉴 아래에 제목이 링크로 뜹니다. 사용을 원치않으실땐 사이드바1에서 "공지리스트"를 빼주시면됩니다.
4. 글작성시 "덧글" 체크박스를 선택하지않으시면 댓글을 달수없도록 아예 글쓰기 폼이 뜨지않습니다.

※ 사용상의 오류에 대해서만 질문받습니다. 제가 일부러 삭제한 기능(방명록/위치로그/트랙백노출/카테고리노출 등)들에 대해 다시 어떻게 추가하느냐...는 식의 원초적인 질문은 사양하겠습니다.
